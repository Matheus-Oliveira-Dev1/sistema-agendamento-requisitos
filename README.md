# PLANEJAMENTO: CLÍNICA MÉDICA

Este planejamento não é nenhum projeto profissional. Sou o Matheus e comecei em 2026 a cursar Ensino Médio Integrado com técnico ( escolhi Técnico de Desenvolvimento de Sistemas ). A motivação desse documento é compartilhar minha evolução, mesmo sendo bem o meu comecinho, é preciso aprender cada vez mais!

*  Neste documento, foi apresentado situação problema para eu e meu grupo de colegas resolver dentro de sala de aula. O tema da aula era sobre requisitos funcionais e não funcionais.

*  Analise de situação-problema dada para nós: praticidade, facilidade e segurança no agendamento de consultas.

# Solução feita pelo nosso grupo:

*  Login facilitado: sem muitas etapas para não ser desafiador e exaustivo;

*  Escolha da modalidade de consulta: um painel com todas opções disponíveis para agendamento e desejo do paciente, ex: clínico geral, pediatria...;

*  Agendamento por calendário e horário: uma tela com calendário e suas opções de datas;

*  Confirmação da consulta: email enviado ao paciente dizendo que a consulta foi agendada com sucesso;

*  Caso de cancelamento: uma mensagem é enviada para o paciente sobre confirmação daquela consulta; se caso cancelado o horário volta a disponibilidade padrão.

*  Segurança: criptografia para proteção de informações dos pacientes e usuários.
